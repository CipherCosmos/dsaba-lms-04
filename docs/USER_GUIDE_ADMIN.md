# Administrator User Guide - DSABA LMS

## System Administration Overview

As a system administrator, you have complete control over the DSABA Learning Management System. This guide covers user management, system configuration, monitoring, and maintenance tasks.

## Getting Started

### Admin Access

1. Log in with your admin credentials
2. Access the admin dashboard
3. Navigate through different administrative modules

### Admin Dashboard

The admin dashboard provides:

- **System Health**: Overall system status
- **User Statistics**: User counts and activity
- **Recent Activity**: System events and logs
- **Quick Actions**: Common administrative tasks

## User Management

### Creating Users

1. Navigate to **Users** > **Create User**
2. Fill in user details:
   - Username and email
   - Personal information
   - Role assignment
   - Department association
3. Set initial password
4. Save user

### Bulk User Creation

1. Go to **Users** > **Bulk Import**
2. Download the user template
3. Fill in user data (up to 1000 users)
4. Upload and validate
5. Confirm import

### User Roles and Permissions

Available roles:

- **ADMIN**: Full system access
- **HOD**: Department head with elevated permissions
- **TEACHER**: Teaching faculty
- **STUDENT**: Regular students

### Managing Departments

1. Access **Settings** > **Departments**
2. Add/edit/delete departments
3. Assign HODs to departments
4. Configure department settings

## Academic Structure Management

### Academic Years

1. Go to **Academic** > **Academic Years**
2. Create new academic years
3. Set start and end dates
4. Mark active academic years

### Batch Management

1. Navigate to **Academic** > **Batches**
2. Create batch instances (classes)
3. Configure:
   - Academic year
   - Department
   - Semester
   - Section
   - Student capacity

### Subject Management

1. Access **Academic** > **Subjects**
2. Create and manage subjects
3. Configure:
   - Subject code and name
   - Credits and type
   - Department association
   - Semester placement

### Subject Assignments

1. Go to **Academic** > **Assignments**
2. Assign teachers to subjects
3. Configure for specific batches
4. Set assignment periods
